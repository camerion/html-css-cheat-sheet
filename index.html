HTML & CSS Cheatsheet

HTML Tags

Tag Name Decription
<table> table creates a table
<tr> table row starts the row for the table
<td> table data houses/hold table data for table cells
<th> table head represents the heading for a coloumn or row
<thead> table heading header of the table
<tbody> table body body of the tsble fits inside
<figure> figure contains the image and caption
<figcaption> figure caption web author can add caption to an image
<a href="mailto:..."> email link href attribute to send an email from users mail service

CSS Properties

Property Description and Values
font-family set specific typeface(use no more than 3 per page)
font-size size of font(px/%/em)
@font-face use if font is not installed on machine
font-weight normal or bold
font-style normal/italic/obique(regular text at an angle)
text-transform upperclass/lowercase/capitalize
text-decoration none, underline, overline, line-through, blink(animate text to flash on and off)
line-height height of entire line of text.best measured in ems, ledding + font-size, makes vertical gap b/w lines of text larger, starting line-height is 1.4 or 1.5
letter-spacing space b/w each letter, use for text-transform uppercase
word-spacing gap b/w words, use if text is bold
